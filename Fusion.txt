Utilisez les mêmes vscode que vous avez utilisé pour l'autre projet mais changez les path : 
--> workspace pour le vscode du backend
--> workspace + build path pour le vscode du frontend
Les autres peuvent rester identiques si vous avez mis des chemins absolus ou si vos dossiers de travails sont exactement au même endroit ceux que vous avez utilisé pour les TP


SQL : 
start.cmd
mysql.cmd
Dedans :

CREATE DATABASE videoclub;
CREATE USER 'videoclub'@'localhost' IDENTIFIED WITH mysql_native_password BY 'videoclub';
GRANT all on videoclub.* TO 'videoclub'@'localhost';
flush privileges;


Dans un terminal vscode à la racine du projet : mvn install
